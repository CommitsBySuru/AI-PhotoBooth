# AI PhotoBooth

An interactive browser-based PhotoBooth application demonstrating both **Vibe Coding** and **Spec-Driven Development (SSD)** workflows using AI-assisted development tools.

---

## Project Overview

This project allows users to:
- Capture photos using their webcam
- Apply real-time visual filters
- Generate a vertical photo strip
- Retry captures
- Download the final photo strip
- Use countdown timer capture
- Experience responsive desktop/mobile UI

The repository demonstrates two different AI-assisted development methodologies through separate Git branches.

---

## Branch Structure

### `vibe_coded_submission`
Demonstrates a rapid and experimental AI-assisted workflow:
- Direct prompting with Codex
- Fast feature implementation
- Minimal planning/documentation
- Added timer functionality
- Added enhanced filters
- Playful UI improvements

### `sdd_submission`
Demonstrates a structured Spec-Driven Development workflow:
- OpenSpec-based planning
- Proposal-driven implementation
- Task tracking and documentation
- Verified UI improvements
- Structured development lifecycle

## Running the Project

1. Clone the repository

```bash
git clone https://github.com/CommitsBySuru/AI-PhotoBooth.git
```

2. Open the project folder

3. Launch `index.html` using:
- Live Server (recommended)
- OR directly in browser

4. Allow webcam permissions
